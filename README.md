# Proyecto: Limpieza de Datos con Programacion Orientada a Objetos

Analisis y limpieza de una encuesta socioeconomica usando Python y una clase personalizada.

## Descripcion

Este proyecto implementa un sistema de limpieza de datos para una encuesta socioeconomica con 50 registros validos.

Se utiliza Programacion Orientada a Objetos (POO) para crear una clase `Limpiador` que:

- Normaliza textos (ciudades)
- Estandariza generos
- Convierte y valida ingresos economicos
- Convierte y valida edades
- Elimina registros duplicados e invalidos
- Genera un resumen descriptivo del dataset limpio

## Objetivos

- Diagnosticar errores en datos reales
- Implementar metodos de limpieza usando POO
- Validar datos contra criterios especificos
- Generar estadisticas descriptivas
